# 개인정보처리방침

**날씨알림카드 (WeatherCard)**

최종 수정일: 2025년 1월 17일

## 1. 수집하는 정보

본 앱은 서비스 제공을 위해 다음 정보를 수집합니다.

### 위치 정보
- 수집 목적: 사용자 위치 기반 날씨 정보 제공
- 수집 항목: GPS 좌표 (위도, 경도)
- 보관 기간: 앱 내 캐시로만 저장, 서버 전송 없음

### 광고 식별자
- 수집 목적: 맞춤형 광고 제공 (Google AdMob)
- 수집 항목: 광고 ID
- 제3자 제공: Google AdMob

## 2. 앱 권한

본 앱은 다음 권한을 사용합니다.

| 권한 | 용도 |
|------|------|
| 위치 (ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION) | 현재 위치 기반 날씨 정보 조회 |
| 다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW) | 화면 잠금 해제 시 날씨 카드 표시 |
| 인터넷 (INTERNET) | 날씨 API 및 광고 데이터 통신 |
| 포그라운드 서비스 (FOREGROUND_SERVICE) | 백그라운드 날씨 업데이트 |

## 3. 정보의 보관 및 파기

- 위치 정보는 기기 내에서만 처리되며 외부 서버로 전송되지 않습니다.
- 앱 삭제 시 모든 데이터가 자동으로 삭제됩니다.

## 4. 제3자 서비스

본 앱은 다음 제3자 서비스를 사용합니다.

- **Google AdMob**: 광고 제공
  - [Google 개인정보처리방침](https://policies.google.com/privacy)
- **기상청 API**: 날씨 데이터 제공

## 5. 아동 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

## 6. 개인정보처리방침 변경

본 방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 공지합니다.
